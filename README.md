# 🔐 Authentication Security Lab

## 📌 Overview
This project is a cybersecurity-based simulation of an authentication system.  
It demonstrates how passwords are created, validated, stored securely using hashing, and tested against brute-force (dictionary) attacks in a controlled environment.

The goal is to understand password security weaknesses and how attackers attempt to exploit them.

---

## 🎯 Features

- Strong password validation (uppercase, lowercase, numbers, symbols)
- Username-based password protection
- SHA-256 password hashing
- Login system with limited attempts
- Account lockout after failed login attempts
- Dictionary-based brute-force simulation
- Password strength analysis using entropy
- Security report generation

---

## ⚙️ Technologies Used

- Python 3
- hashlib (SHA-256 hashing)
- time module
- math module
- re (regular expressions)
